This folder receives the compiled DLLs needed by ECSUnity.
Run build-unity-dll-ecsunity.bat (or build-unity-dll-ecsunity.sh) from the repo root.

Required files (generated by the build step):
  APIFramework.dll         — always required
  APIFramework.pdb         — debug symbols (optional)
  Warden.Contracts.dll     — always required (WorldStateDto lives here)
  Warden.Telemetry.dll     — required for WARDEN builds; omit for RETAIL

DO NOT commit the .dll files to source control.
They are generated from source on each developer's machine.

The .gitignore in ECSUnity/ excludes *.dll from this folder.
